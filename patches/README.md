# patches/

Reserved for unified diffs of the changes to the vendored Tcl/Tk 9.1 and AndroWish
`sdl2tk` sources, generated by reverse-reconstructing the in-place edits from a
pristine `tk9.1b0` + AndroWish `sdl2tk` checkout (as the 8.6 recipe repo does).

Currently the port edits are described in [`../TODO.md`](../TODO.md) §3 and
[`../AGENTS.md`](../AGENTS.md); extracting them into clean, applyable patches here
is an open task.
